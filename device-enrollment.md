# Device Enrollment – Microsoft Intune

## Objective

Enroll Windows device into Microsoft Intune using Microsoft Entra ID join.

---

## Environment

Device: INTUNEWIN11_01  
Join Type: Entra ID Join  
Management: Automatic MDM Enrollment  

---

## Configuration Steps

Device joined through:

Settings → Accounts → Access work or school → Connect
